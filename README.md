## Installation

From Ubuntu 16.04/18.04:

```shell
$ sudo apt-get install g++-4.8 libpcap-dev
$ sudo apt-get update
$ sudo apt-get upgrade

$ git clone https://github.com/Cvilian/TLSniff
$ cd TLSniff
$ sudo make all
```
